== Installation ==
● Make sure you have Woocommerce plugin installed and activated successfully.
● On success, you should see a new item Shipsy Configuration in the menu items on the left side.

== Third-Party Service Usage ==
● This plugin relies on a 3rd party service provided by Shipsy for fetching shipping information. 
**Service URL**: [https://projectxuaeapi.shipsy.io](https://projectxuaeapi.shipsy.io)
**Service Policy**: [https://shipsy.io/privacy-policy](https://shipsy.io/privacy-policy)

● This plugin uses the `libphonenumber-js` library for phone number validation. You can find the source code of the library [here](https://github.com/catamphetamine/libphonenumber-js).
